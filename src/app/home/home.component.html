Welcome to the Cadmus Bibliography Shell. This is the developer base providing the libraries with frontend bibliographic components working with the Cadmus bibliography API.

This shell is here only to provide a playground for the components being developed in the libraries. You can see the topmost component in action, allowing you to pick any works from the database while eventually editing it; see the same component inside a Cadmus part editor; or open the demo page to play with some of the minor components:

  • work picker: a component used to let users pick a work/container.
  • author picker: a component used to let users pick an author.
  • work authors: a component used to let users pick 1 or more authors for a work.
  • keyword picker: a component used to let users pick a keyword.
  • work keywords: a component used to let users pick 1 or more keywords for a work
  • work: a component used to edit a full work data graph, making use of all the preceding components.
  • work filter: a component used to let users define a set of filters for works.
  • work details: a component used to show details about a work or container.
  • work browser: a component used to browse a list of works or containers.
